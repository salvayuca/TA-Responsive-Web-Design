1. Apa itu responsive web design?
=> yaitu suatu keadaan sebuah halaman web dimana tampilannya akan cocok, rapi dan tetap enak dilihat jika diakses dari perangkat apapun dengan resolusi layar yang berbeda. 
2. Sebutkan apa saja relative units yang dimiliki oleh CSS?
=> px, %, rem, em, pt, vw, vh
3. Apa itu breakpoint? Bagaimana cara kita membuat hal tersebut?
=> Breakpoint merupakan titik atau batas dimana konten dari website merespon atau berubah menyesuaikan lebar dari suatu device.
Cara membuat breakpoint yaitu bisa dengan media query, menambahkan max-width atau min-width di CSS
4. Apa yang dapat kita gunakan untuk memanipulasi tata letak suatu element dalam bidang 1 dimensi?
=> flexbox
5. Apa yang dapat kita gunakan untuk memanipulasi tata letak suatu element dalam bidang 2 dimensi?
=> grid
